# Aegis Client

Minimal end-user interface — chat, document upload, status.

## Structure

```
client/
  src/
    components/
    pages/
      Chat/
      DocumentUpload/
      StatusPanel/
    hooks/
    services/
    types/
  public/
  index.html
  vite.config.ts
  package.json
```

## Setup (offline)

```bash
npm install --offline
npm run dev
npm run build
```
